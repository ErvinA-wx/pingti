# 项目收录数据库

`projects.json` 是平替指南自主收录项目的单一数据源。首页“新增平替资源”、`/latest` 最新收录页面，以及上游同步后需要重新插入的本地内容，全部由该文件生成。

## 字段

| 字段 | 必填 | 说明 |
| --- | --- | --- |
| `id` | 是 | 稳定且唯一的站内定位符；发布后不要修改 |
| `name` | 是 | 项目名称 |
| `url` | 是 | 项目官方网站、应用商店或代码仓库地址 |
| `type` | 是 | 资源类型，如 `GitHub`、`网站`、`APP`、`扩展` |
| `category` | 是 | 中文收录分类名称 |
| `categoryHref` | 是 | 对应分类的站内路径 |
| `collection` | 是 | 上游同步后重新插入内容所用的内部集合标识 |
| `addedAt` | 是 | 首次收录日期，格式为 `YYYY-MM-DD` |
| `status` | 是 | 当前状态；公开项目使用 `active` |
| `description` | 是 | 一句话说明项目解决什么问题 |
| `aliases` | 否 | 同一项目的其他网址，用于去重 |

构建与 `npm run local:check` 会校验必填字段和日期格式；构建还会拒绝重复的 `id` 或 `url`。
