# Wiki 更新

- 站点的默认搜索现已改为 **[Exact Match](https://i.ibb.co/rKNb8CGh/34t34g34.png) / [2](https://i.imgur.com/6XUs4fs.png)**，准确度大幅提升。可以在精确匹配和模糊匹配之间切换。感谢 Zenith 完成此项工作。

- 添加了 **[Note Tooltips](https://i.ibb.co/mCZJzj1N/image.png) / [2](https://i.imgur.com/acbozgK.png)**，只需悬停即可查看。感谢 Bread 完成此项工作。

- 现在可以在一次搜索中浏览 [Multiple Result Pages](https://i.ibb.co/5XsSN8bm/image.png)，搜索中已包含页面标题，窗口后方增加了模糊效果。感谢 Bread。

- 添加了 [Index Toggle](https://i.ibb.co/zVhQJDGt/234234234.png) / [2](https://i.imgur.com/oYXk3vw.png)，用于仅查看页面上的索引，类似于星标切换。感谢 Zenith。

- 修复了多个移动端问题，包括输入时搜索不更新；向上滚动时导航栏会立即显示，无需一直滚动到顶部。感谢 Bread。

- 在颜色选择器中添加了 [Monochrome](https://i.ibb.co/CpTJ1yL6/234234.png) / [2](https://i.imgur.com/FN54ocn.png) 作为选项。感谢 Zenith。

- 在游戏工具中添加了 [Xbox Homebrew](https://pingti.org/gaming-tools#xbox-homebrew) 部分。

- 在游戏工具中添加了 [Undertale / Deltarune](https://pingti.org/gaming-tools#undertale-deltarune) 部分。感谢 Nattakit、Eitan、Chara 和 Clara 的贡献。

- 在游戏工具中添加了 [RimWorld Tools](https://pingti.org/gaming-tools#rimworld-tools) 部分。感谢 Litekin 的贡献。

- 在教育分类下添加了 [Aerospace Engineering](https://pingti.org/educational#aerospace-engineering) 部分。感谢 Yeet + Manu 的贡献。

- 添加了 [Indie Browser Games](https://pingti.org/gaming#indie-browser-games) 部分，并为 Nitrome 加星。

- 添加了 [Unblocked Browser Games](https://pingti.org/gaming#unblocked-games) 部分，并为 Selenite 加星。

- 在健康分类下添加了 [HRT / 跨性别健康](https://pingti.org/misc#hrt-trans-health) 部分。感谢 April + Pluey 的贡献。

# 已添加星标 ⭐

- 在游戏启动器中为 [Hydra](https://pingti.org/gaming-tools#game-launchers) 加星。支持基于插件的下载、BT 下载、RD + TorBox、良好的自定义、成就、好友列表、游戏时间监控、云存档等。

- 在 Android 音频中为 [SpotiFLAC-Mobile](https://pingti.org/mobile#android-audio) 加星。开源下载应用，支持多个站点。

- 在 MC 启动器中为 [ElyPrismLauncher + Freesm Launcher](https://pingti.org/gaming-tools#launchers) 加星。两者都是 Prism 的功能丰富分支，在我们尝试过的其他分支中脱颖而出。

- 在 AI 聊天机器人中为 [Gemini](https://pingti.org/ai#official-model-sites) 加星。Gemini 3 Flash 现在运行良好，在 LMArena 上排名靠前。

- 在 Android YouTube + Android YouTube 音乐中为 [Morphe](https://pingti.org/mobile#android-youtube-apps) 加星。无广告的 YouTube 修补程序 / ReVanced 替代品。

- 在视频下载中为 [XDMovies + The One Piece](https://pingti.org/video#download-sites) 加星。两者都有庞大的资源库和快速的主机。

- 在 Android 视频播放器中为 [mpvEx](https://pingti.org/mobile#android-video-players) 加星。功能丰富的 mpv 分支，改进的 UI，维护良好。

- 在游戏下载中为 [Virgil Game Search](https://pingti.org/gaming#download-games) 加星。自定义多站点游戏下载搜索引擎，仅使用可信来源。

- 在免费软件站点中为 [Dan Store](https://pingti.org/downloading#freeware-sites) 加星。基于网页的 Microsoft Store 浏览器，无需 Microsoft Store 应用。

- 在电子书阅读器中为 [KoReader](https://pingti.org/reading#ebook-readers) 加星。开源、高度可定制、支持插件，非常适合电子阅读器。

- 在电影推荐工具中为 [Nothing To Watch](https://pingti.org/video#recommendation-tools) 加星。交互式发现可视化，一种有趣且独特的方式来寻找可观看的内容。

- 在地理分类中为 [Ekvis](https://pingti.org/educational#geography-sociology) 加星。维护良好且实用的地理学习网站，涵盖 220 多个国家、旗帜、领土等。

- 在浏览器游戏中收藏了 [Slope Plus](https://pingti.org/gaming#browser-games)。Slope 的更新版本；具有额外功能、游戏模式以及活跃的自定义地图社区。

- 在图像与视频生成中收藏了 [GeminiGen AI](https://pingti.org/ai#video-generation)。提供无限的 Nano Banana Pro 图像，以及无限的 Sora 2、Veo 3.1 和 Grok 视频。

- 收藏了 [Perchance](https://pingti.org/ai#image-generation) 图像生成。无限使用，无需注册，且质量提升了很多。

- 在 Lo-Fi 电台中收藏了 [Flow-Fi](https://pingti.org/audio#lofi-radio)。提供多种电台和视频背景选项；比大多数收到的 lo-fi 网站质量更高。

- 在导航部分收藏了 [StreetComplete](https://pingti.org/misc#navigation-transport)。通过完成任务帮助填补缺失的 OpenStreetMap 数据。

- 在浏览器游戏中收藏了 [Flash Arch、Kongregate 和 Crazy Games](https://pingti.org/gaming#multi-game-sites)。浏览完这部分后，这三个似乎最突出，但如果遗漏了其他好的，请随时告知。我们也考虑过 AddictingGames 和 Poki，但对两者都不太确定。
