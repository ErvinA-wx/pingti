# 已添加星标 ⭐

- 在浏览器隐私部分为 [LibreWolf](https://fmhy.net/privacy#browser-privacy) 添加星标。基于 Firefox 的定制隐私版本，易于获取，配置良好且广受好评。

- 在邮件隐私部分为 [Tuta](https://fmhy.net/privacy#email-privacy) 添加星标。加密邮件，提供 1GB 免费空间，部分人认为优于 Proton Mail 等产品。

- 在移动端 Discord 客户端部分为 [Kettu](https://fmhy.net/mobile#social-media-apps) 添加星标。同时支持 Android 和 iOS 的修改版客户端，支持插件，测试者反馈良好。

- 在 Android 在线观看部分为 [Nuvio](https://fmhy.net/mobile#android-streaming) 添加星标。该应用已在 iOS 部分获得星标，但 Android 用户似乎也觉得它很有用。支持插件（包括 Stremio），界面美观，每周更新。

- 在 AI 部分为 [Qwen](https://fmhy.net/ai#official-model-sites) 和 [Ask Brave](https://fmhy.net/ai#specialized-chatbots) 添加星标。两者均无使用限制，且模型表现良好。

- 在隐私索引部分为 [The OPSEC Bible](https://fmhy.net/privacy#privacy-indexes) 添加星标。内容深入且备受推崇的在线匿名指南。

- 在 FOSS 部分为 [Is it really FOSS](https://fmhy.net/downloading#foss-sites) 添加星标。帮助判断某项目是否真正开源的数据库。

- 在用户脚本管理器部分为 [ScriptCat](https://fmhy.net/internet-tools#userscripts) 添加星标。开源，兼容 MV3，支持后台和定时脚本，被部分人视为当前最佳脚本管理器。

- 在电子书阅读器部分为 [Readest](https://fmhy.net/reading#ebook-readers) 添加星标。可定制，支持多种格式和平台，提供带进度同步的免费云存储。

- 在浏览器游戏网站部分为 [Flash Museum](https://fmhy.net/gaming#multi-game-sites) 添加星标。界面简洁，拥有较大的老式 Flash 游戏库。

- 在付费墙绕过部分为 [PressReleased](https://fmhy.net/internet-tools#paywall-bypass) 添加星标。基于 Bypass Paywalls Clean，无需安装扩展。

- 在重定向绕过部分为 [Evade](https://fmhy.net/internet-tools#redirect-bypass) 添加星标。速度快，维护良好，适用于大量网站。

- 在词典部分为 [Wiktionary](https://fmhy.net/educational#dictionaries-thesaurus) 添加星标。备受推崇的词典，拥有众多活跃贡献者。

- 在 VPN 部分为 [Mullvad + IVPN](https://fmhy.net/privacy#vpn) 添加星标。两者均已确认 [No-Logging](https://mullvad.net/en/blog/2023/4/20/mullvad-vpn-was-subject-to-a-search-warrant-customer-data-not-compromised/) / [2](https://www.ivpn.net/knowledgebase/privacy/how-do-we-react-when-requested-by-an-authority-for-information-relating-to-a-customer/)，支持多种支付方式，无需邮箱。

- 在代理站点中重新加星了 [HolyUnblocker](https://fmhy.net/privacy#proxy-sites)。再次正常工作，内置广告拦截和 Tor 支持。

# 已移除的内容

- 移除了 P-Stream，因为其所有者决定[退出该项目](https://i.ibb.co/6RRB4YBj/image.png) / [2](https://i.imgur.com/xYIO9aK.png)。

- 移除了 Tampermonkey，因为多年来它在隐私政策、闭源、更新导致数据丢失等方面存在各种问题，而且有更好的替代方案（ScriptCat、Violentmonkey、Firemonkey）。

- 移除了 RTINGS，因为它们已[完全付费](https://www.rtings.com/company/revamping-our-membership-program)。

- 取消了 ReVanced 的星标，因为 Morphe 现在足够稳定可以替代它，并且开发更活跃。

- 取消了 Trakt 的星标，因为它们现在[强制用户使用 v3 设计](https://redd.it/1rwgf8y)，与之前版本相比缺少许多功能。
