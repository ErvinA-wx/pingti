# 已添加星标 ⭐

- 在文档部分为 [Epstein Exposed + JMail](https://fmhy.net/reading#declassified-leaked-documents) 添加了星标，我们收到了很多此类推荐，但这两个的界面似乎更好。

- 在文件转换器中为 [Convert to it!](https://fmhy.net/file-tools#file-converters) 添加了星标，几乎支持所有可想象的格式，文件上传仅在浏览器中进行，不会上传到他们的服务器，有利于隐私保护。

- 在文件托管站中为 [ZeroFS](https://fmhy.net/file-tools#file-hosts) 添加了星标，文件大小无限制，文件在最后一次下载后保留 30 天。

- 在未封锁浏览器游戏中为 [GN-Math](https://fmhy.net/gaming#unblocked-games) 添加了星标，包含许多优质移植作品，似乎很受欢迎。

- 在字体工具中为 [Font Analyzer](https://fmhy.net/text-tools#font-tools) 添加了星标，可以分析任何页面的字体并提供类似的替代方案。

- 在 Pastebin 中为 [Markbox](https://fmhy.net/text-tools#pastebins)（又名 fluffle）添加了星标，是 rentry 的替代品，界面类似，支持 Markdown、编辑等功能。

- 在图像编辑中为 [Efecto](https://fmhy.net/image-tools#image-effects) 添加了星标，基于网页的视觉效果编辑器，支持图像、视频和 3D 模型。

- 在视频生成中为 [KLIPY](https://fmhy.net/ai#video-generation) 添加了星标，可以使用 Veo 3 生成无限 GIF。

- 为 [Image-Upscaling](https://fmhy.net/image-tools#upscale-restore) 添加了星标，在线图像放大工具，运行快速，效果良好。

- 在图像生成中为 [Google Flow](https://fmhy.net/ai#image-generation) 添加了星标，提供无限的 Nano Banana Pro / Nano Banana 2。

- 在图像生成中为 [Khoj](https://fmhy.net/ai#image-generation) 添加了星标，提供无限的 Nano Banana / Imagen 4。

- 在教育技能中标记了 [Knots 3D](https://fmhy.net/educational#skills-hobbies-diy)，提供 3D 交互指南，学习如何打各种绳结。

- 在专业聊天机器人中标记了 [Google AI Mode](https://fmhy.net/ai#specialized-chatbots)，AI 搜索引擎，提供无限制的 Gemini 3。

- 在审查绕过版块中标记了 [Net4People](https://fmhy.net/privacy#anti-censorship)，全球审查规避讨论，聊天在 GitHub 问题版块内进行。

- 在 VPN 服务器中标记了 [NetBird](https://fmhy.net/privacy#vpn-server)，WireGuard Mesh 类似 Tailscale，但开源、可自托管且具有 Web UI。

- 在 RSS 阅读器中标记了两个 [FreshRSS](https://fmhy.net/internet-tools#rss-readers) 实例（rss.fmhy + Prodottoinrete），可自定义、基于 Web，支持无限订阅源。
