# Wiki 更新

- 添加了 FMHY 的 **[Alternative Frontend](https://ffmhy.pages.dev/)**，采用完全不同的设计。该版本从官方源拉取，因此会与新编辑保持同步。还提供了一个每页有效的[随机站点](https://i.ibb.co/fVkHqhRP/image.png) / [2](https://i.imgur.com/p4Mxs8y.png) 按钮。注意，该版本的前端页面已被移除，现在直接指向 wiki 本身。感谢 nw 制作此版本。

- 在备份页面中添加了 **[3 New Instances](https://fmhy.net/other/backups)**（Samidy、JBugel、ArtistGrid）。

- 在颜色选择器中添加了 **[Catppuccin](https://i.ibb.co/Ps8vDJL0/image.png)** / [2](https://i.imgur.com/558l4gH.png) 作为选项。感谢 Samidy 完成此工作。

- 为 [Textbooks](https://fmhy.net/reading#textbooks) 添加了新板块。

- 为[文件信息/元数据](https://fmhy.net/file-tools#file-info-metadata)添加了新板块。

- 将 Linux 和 Mac BT 客户端合并到 [Main Section](https://fmhy.net/torrenting#torrent-clients)，并添加了标签和平台图标。[Before vs. After](https://i.ibb.co/fV1zdt44/Untitled.jpg) / [2](https://i.imgur.com/THLiMqL.jpeg)

- 根据文库大小和界面重新排序了 [Light Novel Sites](https://fmhy.net/reading#light-novels)，添加了标签，移除了小型文库，并收藏了两个突出的站点（NovelCool + WuxiaClick）。[Before vs. After](https://i.ibb.co/DSRfsrM/Untitled.png) / [2](https://i.imgur.com/IMRUQoo.png)

- 精简了 [Audio Recommendation](https://fmhy.net/audio#curated-recommendations) 板块，并为[歌曲/艺术家发现](https://fmhy.net/audio#song-artist-discovery)添加了新板块。[Before vs. After](https://i.ibb.co/zTDNZgr9/Untitled.png) / [2](https://i.imgur.com/V6h0mQx.png)

- 为 [Linux Cheat Sheets](https://fmhy.net/linux-macos#cli-cheat-sheets) 和 [Linux Distros](https://fmhy.net/linux-macos#linux-distros) 添加了标签。感谢 [helxop](https://github.com/fmhy/edit/commit/9a9cd6dd47027ac63370b453ec86a943cdc0b9d6)。[Before vs. After](https://ibb.co/gMD8mKbw) / [2](https://i.imgur.com/hqqLsCB.png)

- 改版了 [Feedback Window](https://i.ibb.co/JjpzRXL7/image.png) / [2](https://i.imgur.com/nODLPVM.png)，以改善其外观。感谢 Samidy 完成此工作。

- 为 [Grading Page](https://fmhy-grading.pages.dev/) 添加了备份/前端，并将 [Backup](https://fmhy.net/other/backups) 和 [FAQ](https://fmhy.net/other/FAQ) 页面移至网站，而非 Reddit/GitHub。感谢 Roi Goat 和 Samidy 完成此工作。

- 改进了[主题处理器](https://github.com/fmhy/edit/pull/4386)，这将有助于将来更轻松地切换或添加主题（如万圣节、圣诞节等）。感谢 Land 完成此工作。

- 将 360 Total Security 添加到不安全站点。如果安装其任何应用，它会不断弹出提示安装“工具箱”，或者在使用清理、修复或优化选项时未经同意安装。安装后，工具箱会[修改默认应用](https://en.wikipedia.org/wiki/Criticism_of_Qihoo_360#Malicious_promotion)（如浏览器），并将其全部切换为 360 选项。

# 已添加星标 ⭐

- 在视频下载中标记了 [DDLBase](https://fmhy.net/video#download-sites)。同时提供 4K 和 1080p 内容，仅使用快速文件托管站，且拥有庞大的资源库。

- 在动漫在线观看中标记了 [Anime Realms](https://fmhy.net/video#anime-streaming)。来源优质，界面美观，支持自动连播、自定义播放器和多语言。

- 在 Telegram 音频机器人中标记了 [Music Hunters](https://fmhy.net/audio#telegram-bots)。支持众多网站，音质良好，易于使用。

- 在游戏下载中标记了 [AstralGames](https://fmhy.net/gaming#download-games)。资源库庞大，文件托管站快速，提供预安装版本，并允许请求。

- 在漫画网站中标记了 [Comix](https://fmhy.net/reading#manga)。资源库庞大，允许上传，界面美观，即将增加更多语言。

- 在音频在线观看中标记了 [Monochrome](https://fmhy.net/audio#streaming-sites)。无需注册，支持多 API 集成、Last.fm 集成，可自定义，资源库庞大。

- 在在线观看中标记了 [Rive, Aether, Willow, and FlyX](https://fmhy.net/video)。这些网站拥有出色的自定义播放器、丰富功能、快速流媒体和令人印象深刻的界面。

- 在体育直播中标记了 [StreamSports99](https://fmhy.net/video#live-sports)。服务器快速，界面稳固。

- 在 Twitter/X 自定义中标记了 [Control Panel for Twitter](https://fmhy.net/social-media-tools#twitter-x-customization)。增加了许多额外功能，可隐藏不使用的部分，改进界面等。

- 在文件托管站中标记了 [Rootz](https://fmhy.net/file-tools#file-hosts)。速度良好，单文件最大 25GB，最后一次下载后保留 15 天。

- 在图像生成中标记了 [PigenAI](https://fmhy.net/ai#image-generation)。提供无限制的 Imagen 4、Qwen 和 Nano Banana。

- 在 iOS 在线观看中标记了 [Nuvio](https://fmhy.net/mobile#ios-streaming)。使用 Stremio 插件，限制少于当前 iOS 上可用的“Lite”版本。

- 在 Android BT 下载中标记了 [Flud](https://fmhy.net/mobile#android-torrenting)。允许绑定，维护良好，广告可通过 DNS 轻松屏蔽。

- 在身体健康中标记了 [Sleeping Guide](https://fmhy.net/misc#physical-health)。一份睡眠卫生/科学指南；目前反响良好，由 Dan 策划。

- 在重制/复刻中标记了 [OpenMW](https://fmhy.net/gaming#remakes-recreations)。对 Morrowind 的完全从头重实现，[approved by Bethesda](https://wiki.openmw.org/index.php?title=Bethesda_Emails) 本身。

- 在重制/复刻中标记了 [Clone Hero](https://fmhy.net/gaming#remakes-recreations)。功能丰富的 Guitar Hero 克隆，数千首歌曲和播放列表，无延迟，支持所有吉他控制器和 MIDI 鼓组。

- 在多人浏览器游戏中标记了 [Gidd](https://fmhy.net/gaming#party-multiplayer)。提供高质量的大富翁、GeoGuessr、卡牌游戏等版本。

- 在虚拟机中标记了 [QEMU](https://fmhy.net/system-tools#virtual-machines)。在非 Windows 系统上性能良好；建议配合 Virt-Manager 或 Vagrantup 使用。

- 在文件备份中标记了 [Restic](https://fmhy.net/file-tools#file-backup)。易于使用，跨平台，在基准测试中表现良好。

- 在编程 AI 中标记了 [Gemini CLI](https://fmhy.net/ai#coding-ais)。目前似乎拥有最好的限制之一。

# 已移除的内容

- 移除了 CrocDB，因其已遭到 [hit by DMCA](https://i.ibb.co/RpDvJFgf/image.png) / [2](https://i.imgur.com/iSV0MHq.png)。

- 移除了 IcebergCharts 网站，因为他们已被 [hacked a few times now](https://i.ibb.co/XxMMwqDk/image.png) / [2](https://i.imgur.com/MkvElvr.png)，且无法查明原因。如果问题解决，未来可能会重新添加。注意，目前会保留其 subreddit。

- 移除了 CanvasBlocker，因其仅支持 Firefox，而 FF 现已内置此功能；此外，在某些情况下它可能反而使指纹识别更差，且已不再积极维护。

- 从“体育直播”中移除了 FTV 和 WAC，因为两者均已下线。

- 移除了 Foxified，因为它突然[用广告填充结果](https://i.ibb.co/4RzPsQ8Z/image.png) / [2](https://i.imgur.com/AjBMdfX.png)。

- 取消了对 OnTheSpot 的标星，因为它即将失去 Spotify 支持。

- 在“编程 AI”中取消了对 Windsurf 的标星，因为其限制和免费模型已不再那么好用。
