大家好，新年快乐！告别 '23 之际，让我们花点时间回顾一下 FMHY 今年走过的非凡历程：

- 我们制作了一个新网站，配备了一系列精心设计的功能（当然是我做的），以提升体验，让在所有设备（包括移动设备）上浏览 wiki 都变得愉快。
- 我们开放了 Discord 服务器，成员达到 1 万，为社区带来了许多优秀的新人，并使贡献变得容易。
- 我们在 Reddit 上已达到 25 万+ 成员和 3100 万次浏览量！
- 我们的 FMHY 编辑仓库见证了社区提交的 1000+ 拉取请求，并合并了 8000+ 次提交！
- 这个 wiki 已经发展得不仅仅是盗版索引，所以我们觉得是时候让其他部分也得到一些关注了。因此，我们整理并将工具部分分离到各自的页面，从而将许多有用的工具从存储中移出，使其更容易被发现。
- 我们开始了 FMHY 的精简阶段，从 2.7 万个链接减少到 2.4 万个！我们移除了许多失效、无用和冷门的链接/类别，并重新组织了许多部分，以提高清晰度和导航便利性。

进入 2024 年，我们热切期待在已有成就的基础上继续发展，共同进步。没有社区中每个人的爱与支持，FMHY 就不会有今天。感谢大家成为 FMHY 不可或缺的一部分。让我们迎接又一个成长与成功之年！💙

  [技术工作](/misc#tech-jobs)、[初创公司](/misc#startup)和

### 已添加星标 ⭐

- 在在线观看站点中为 [Braflix](/video#streaming-sites) 添加星标。流媒体速度快、无广告、界面美观、支持 4K 和 1080p。

- 在阅读站点中为 [Liber3](/reading#ebooks) 添加星标。资源库庞大、界面美观、一键下载。

- 在视频播放器中为 [Screenbox](/video-tools#video-players) 添加星标。基于 VLC 的开源视频播放器，界面美观。

- 在 Discord 客户端中为 [Vesktop](/social-media-tools#discord-clients) 添加星标。基于 Vencord 的轻量桌面客户端。

- 在 AI 聊天机器人中为 [SydneyQt](/ai#proprietary-llms) 添加星标。越狱版 Bing AI。

- 在 FOSS APK 中为 [App Manager](/mobile#foss-apks) 添加星标。开源的 Android 应用包管理器。

- 在 Android 工具中为 [scrcpy](/mobile#android-device) 添加星标。通过桌面管理移动设备。

- 在 Android 设备中为 [KernelSU](/mobile#android-device) 添加星标。部分用户更偏好的 Root 工具。

- 在 iOS 隐私中为 [Onion Browser](/mobile#ios-privacy) 添加星标。Tor 官网推荐。

- 在自制软件中为 [Wii Guide](/gaming#homebrew) 添加星标。

- 在视频工具中为 [Video2x](/video-tools#video-tools-1) 添加星标。无损视频/GIF/图像放大工具。

- 在 Windows 更新中为 [VCRedist](/system-tools#windows-updates) 添加星标。轻松下载所有 Microsoft Visual C++ 可再发行组件。

- 在隐私扩展中为 [CanvasBlocker](/privacy#privacy-extensions) 添加星标。防止 JavaScript API 指纹识别。

### 已移除的内容

- 移除了 SevenGamers，因其
  [决定关闭](https://i.imgur.com/C5sgVqI.png)。

- 移除了 YasDL，因其上传了包含恶意软件的 StarDock 版本，与 filecr 等被移除的原因相同。

- 移除了 Torrentz2，因其已停止索引新种子。

- 取消了 LookMovie 的星标，因其在视频播放前增加了
  [10 秒等待](https://i.imgur.com/I0D9Hyt.png)。

- 在游戏工具中取消收藏 DS4Windows。只需将游戏添加到 Steam，然后右键点击游戏即可开启 PS4 控制器支持。

- 取消收藏 Ideogram，因其不再提供无限使用。

- 移除 EveryNoiseAtOnce，因为开发者已被 [Spotify 解雇](https://i.imgur.com/AaIrcAc.png)。

- 移除 CryptoStorm，因其 [可能存在安全隐患](https://i.imgur.com/VDlSY1T.png)。所有者显然已不再与其有关联，但一家有着如此黑暗历史的公司仍然难以信任。

- 从 NotABug 移除备份。该站点目前几乎 [每天](https://i.imgur.com/Vx8Ou68.png) 都会宕机，要在上面维护 25 个 wiki 页面简直是一场噩梦。如果他们能修复问题，我们可能会恢复备份。
