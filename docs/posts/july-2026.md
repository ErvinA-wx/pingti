# 已添加星标 ⭐

- 在 Android YouTube 音乐应用中为 [ArchiveTune](https://fmhy.net/mobile#youtube-music) 添加星标，界面美观且运行良好。

- 在食品板块为 [Too Good To Go](https://fmhy.net/misc#food) 添加星标，获取廉价食物并减少浪费的好方法，试用过的成员反馈良好。

- 在阅读追踪/数据库中为 [Hardcover](https://fmhy.net/reading#tracking-database) 添加星标，界面美观、功能丰富，并支持从其他平台（如 goodreads）导入。

- 在 Pastebins 中为 [Yasosu](https://fmhy.net/text-tools#pastebins) 添加星标，支持 Markdown、可编辑帖子，并支持自定义 URL。

- 在精选推荐中为 [The Criterion Closet](https://fmhy.net/video#curated-recommendations) 添加星标，3D Criterion 步入式衣橱复刻，带筛选功能。

- 在 Linux 音频中为 [LibrePods](https://fmhy.net/linux-macos#linux-audio) 添加星标，可在 Linux（以及已 root 的安卓）上使用 AirPod 功能。

- 在动漫 BT 下载中为 [SeaDex](https://fmhy.net/video#anime-torrenting) 添加星标，动漫视频质量指南，可配合 Stremio + AIOStream 使用。

- 在 Linux 系统中为 [LACT](https://fmhy.net/linux-macos#linux-system) 添加星标，功能丰富的 GPU 配置与监控，简化超频。

- 在机器学习中为 [Unsloth](https://fmhy.net/ai#machine-learning) 添加星标，易于使用的微调 + AI 套件，广受好评。

- 在元素周期表中为 [Zperiod](https://fmhy.net/educational#periodic-tables) 添加星标，简洁的交互式周期表，附带 3D 原子模型。

- [Bookracy](https://fmhy.net/reading#ebooks) 已恢复，已在电子书板块重新添加并加星。

- [Soft98 + DownloadHa](https://fmhy.net/downloading#software-sites) 均已恢复运行，已在软件板块重新添加并加星。
