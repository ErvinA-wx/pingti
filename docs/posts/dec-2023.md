### 已添加星标 ⭐

- 在在线观看站点中为 [Streamflix](/video#multi-server) 添加星标。界面
  美观，1080p 快速，由社区成员运营。

- 在在线观看站点中为 [watch.lonelil](/video#multi-server) 添加星标。
  界面美观，4K + 1080p 快速，提供直播电视。

- 在游戏下载中为 [Game Bounty](/gaming#download-games) 添加星标。
  快速文件托管站，预安装，干净上传，由社区成员运营。

- 在废弃软件游戏中为 [AbandonwareGames](/storage#abandonware-games) 添加星标。
  界面美观，拥有大量老游戏库。

- 在 Steam / Epic 中为 [Heroic Games Launcher](/gaming#steam-epic) 添加星标。
  开源的 Epic 游戏启动器。

- 在下载目录中为 [TG Archive](/downloading#download-directories) 添加星标。
  Telegram 文件存档 + 搜索工具。

- 在系统精简中为 [BleachBit](/system-tools#system-debloating) 添加星标。
  流行的系统文件清理工具。

- 在 YouTube 下载中为 [YT-DLP-GUI](/video-tools#youtube-download) 添加星标。
  适用于 YT-DL 的出色图形界面，支持一键安装。

- 在系统调整中收藏了 [EverythingToolbar](/system-tools#system-tweaks)。
  为 Windows 工具栏增加了许多功能。

- 在翻译工具中收藏了 [DeepL](/text-tools#translators)。人们认为这
  能提供一些最佳结果。

- 在游戏优化中收藏了 [Magpie](/gaming#optimization-tools)。
  在任何游戏或设备上启用 AMD FSR。

- 在 iOS 应用中收藏了 [TrollStore](/mobile#ios-apps)。无需电脑、付费开发者账户或越狱即可永久
  安装非 App Store 的 iOS 应用。

- 在智能电视 / Firestick 中收藏了 [SmartTube](/video#smart-tv-firestick)。
  无广告的 YouTube 应用。原名 SmartTubeNext。

### 已移除的内容

- 移除了 NovaAI，因为他们
  [决定关闭](https://www.reddit.com/r/Piracy/comments/17pzrzj/nova_oss_the_api_that_provided_free_gpt4_and/)。

- 移除了 LocalCDN 和 Decentraleyes，因为在启用
  [Total Cookie Protection](https://blog.privacyguides.org/2021/12/01/firefox-privacy-2021-update/#localcdn-and-decentraleyes)
  后它们已冗余。

- 将所有 CurseForge 链接替换为 Modrinth，因为人们认为 CurseForge
  [与用户群体合作不佳](https://youtu.be/Vhdwz5apiQQ?si=xgzkQFa1S7hZNa5-)。

- 取消收藏 12ft.io，因为它不再能绕过非 JavaScript 的付费墙。

- 移除了 FileListing，因为他们已关闭服务器。

- 移除了 GreaseMonkey，因为它已三年未更新，并且可能有
  跟踪用户的历史。
