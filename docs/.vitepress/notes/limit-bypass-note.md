#### 限制绕过说明

Apple 仅允许使用免费 Apple 账户侧载 3 个应用（包括 SideStore）。要在 SideStore 上绕过此限制，可以使用 [SparseBox](https://docs.sidestore.io/docs/advanced/alternative#sidestore-exploits) / [指南](https://docs.sidestore.io/docs/advanced/alternative#sidestore-exploits)（iOS 17.0 - 18.1 Beta 4，不包括 17.7.1、17.7.2）或 [LiveContainer](https://github.com/LiveContainer/LiveContainer)（iOS 16+）。
