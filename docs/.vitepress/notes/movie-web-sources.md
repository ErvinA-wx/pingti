#### 为 P-Stream（及所有 movie-web 分支）添加源

可以启用一个扩展：[Firefox](https://addons.mozilla.org/firefox/addon/p-stream-extension-v1/) / [Chrome](https://chromewebstore.google.com/detail/p-stream-enchanced/geimjiobahmbkmaigaafjnoanebpmlmj) / [script](https://github.com/xp-technologies-dev/userscript)，它会添加更多源，但需要连接到所有网站才能运行。该扩展是安全的，很多人都在使用，这些权限只是为了扩展能正常工作。

请注意，如果不想使用主浏览器，可以在新浏览器或新的浏览器配置文件中运行。
